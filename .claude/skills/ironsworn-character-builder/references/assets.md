# Ironsworn Assets — Character Creation Reference

78 assets across 4 categories. At character creation, choose 3 from any categories. Each asset's first ability (●) is active immediately; the others (○) unlock by spending experience later.

> **When presenting assets:** Show one category at a time. If the player described a concept, filter to the 6–10 most relevant options first before showing the full list. State the starting ability in plain language — don't just quote mechanics verbatim.
